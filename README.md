# Auditoría de Accesibilidad Web: FAGDUT (Asociación Gremial de Docentes de la UTN)

[cite_start]Este repositorio contiene el análisis de accesibilidad realizado sobre el sitio web de **FAGDUT**, desarrollado como Trabajo Práctico para la asignatura **Diseño Inclusivo para Usuarios con Discapacidad**[cite: 6]. 

## 🎯 Objetivo y Marco Normativo
[cite_start]El objetivo fue verificar el cumplimiento de los **38 criterios de conformidad** de accesibilidad establecidos en la **Disposición 6/2019 (Anexo II)** para sitios web de alcance nacional en Argentina[cite: 21, 47]. 

[cite_start]Aunque la consigna general de la cátedra mencionaba otros sitios de la universidad, este análisis se centró específicamente en las secciones críticas de la plataforma de **FAGDUT**[cite: 21].

## 🛠️ Metodología y Herramientas
[cite_start]Siguiendo los requisitos de aprobación, se empleó una combinación de testeo automático y manual[cite: 48]:
* [cite_start]**Validación Automática:** AXE y WAVE[cite: 24].
* [cite_start]**Análisis Visual:** Colour Contrast Analyser (contraste de color) y pruebas de Zoom[cite: 25, 29].
* [cite_start]**Navegación:** Pruebas exclusivas mediante Teclado[cite: 30].
* [cite_start]**Tecnología Asistiva:** Uso del lector de pantalla **NVDA** para validar el orden de lectura y la robustez del sitio[cite: 31].

## 📂 Documentación del Proyecto
La evidencia detallada se encuentra organizada en la carpeta [/docs](./docs/):
* [cite_start]**[Grilla de Reporte de Errores](./docs/GRILLA%20REPORTE%20errores%20Accesibilidad.xlsx):** Informe técnico que vincula cada error con su criterio WCAG, incluye capturas de pantalla como evidencia y proporciona recomendaciones de mejora[cite: 49, 50, 51, 52, 53].
* [cite_start]**Guía de Normas:** Documentación de apoyo basada en las **WCAG 2.2** para comprender los niveles de conformidad[cite: 38].

## 📈 Conclusiones
[cite_start]El trabajo identifica fortalezas y áreas de mejora prioritarias, especialmente en aspectos de contraste y etiquetado de imágenes, fundamentales para garantizar la inclusión digital en el ámbito gremial docente[cite: 56, 57, 58].
