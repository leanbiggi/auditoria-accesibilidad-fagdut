# Auditoría de Accesibilidad Web: FAGDUT (Asociación Gremial de Docentes de la UTN)

Este repositorio contiene el análisis de accesibilidad realizado sobre el sitio web de **FAGDUT**, desarrollado como Trabajo Práctico para la asignatura **Diseño Inclusivo para Usuarios con Discapacidad**.

## 🎯 Objetivo y Marco Normativo
El objetivo fue verificar el cumplimiento de los **38 criterios de conformidad** de accesibilidad establecidos en la **Disposición 6/2019 (Anexo II)** para sitios web de alcance nacional en Argentina. 

Aunque la consigna general de la cátedra mencionaba otros sitios de la universidad, este análisis se centró específicamente en las secciones críticas de la plataforma de **FAGDUT**.

## 🛠️ Metodología y Herramientas
Siguiendo los requisitos de aprobación, se empleó una combinación de testeo automático y manual:
* **Validación Automática:** AXE y WAVE.
* **Análisis Visual:** Colour Contrast Analyser (contraste de color) y pruebas de Zoom.
* **Navegación:** Pruebas exclusivas mediante Teclado.
* **Tecnología Asistiva:** Uso del lector de pantalla **NVDA** para validar el orden de lectura y la robustez del sitio.

## 📂 Documentación del Proyecto
La evidencia técnica detallada se encuentra organizada en las siguientes secciones:
* **[Grilla de Reporte de Errores](./docs/GRILLA%20REPORTE%20errores%20Accesibilidad.xlsx):** Informe técnico que vincula cada error con su criterio WCAG, incluye capturas de pantalla como evidencia y proporciona recomendaciones de mejora.
* **[Guía de Normas](./docs/4-NORMAS%20WCAG2.2-para%20comprenderlas.pdf):** Documentación de apoyo basada en las **WCAG 2.2** para comprender los niveles de conformidad.
* **[Evidencias Visuales](./evidencias/):** Capturas de pantalla de los fallos detectados durante el proceso de testing.
* **[Listado de Herramientas](./herramientas/herramientas.md):** Detalle técnico de los softwares y metodologías aplicadas durante la auditoría.

## 📈 Conclusiones
El trabajo identifica fortalezas y áreas de mejora prioritarias, especialmente en aspectos de contraste y etiquetado de imágenes, fundamentales para garantizar la inclusión digital en el ámbito gremial docente.
